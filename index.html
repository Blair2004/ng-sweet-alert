Ng-sweet-alert

Sweet alert directive for angular js, without writing single javascript code

View on GitHub Download .zip Download .tar.gz

Welcome to Ng-sweet-alert.

This Directive can be use in angular js application ,without writing a javascript code,All configuration can be done in HTML itself

Getting Started

Install it via bower

bower install ng-sweet-alert

and add ng-sweet-alert as dependency in your main module:

angular.module('yourApp',[
  'ng-sweet-alert'
]);

You can try by chaging option on fly and try it!!! Enjoy

Option
sweet-options
sweet-confirm-option
sweet-cancel-option
sweet-on-confirm checkConfirm()
sweet-on-cancel checkCancel()
